# Loan Data Analysis

## 📊 Project Overview
This project focuses on analyzing loan data to understand patterns related to income, loan amount, and loan purpose.

## 📁 Dataset
The dataset was taken from Kaggle and contains information about:
- Age
- Gender
- Education
- Income
- Loan Amount
- Loan Intent

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Steps Performed
1. Data Loading using Pandas
2. Data Cleaning
3. Data Exploration (df.head(), df.info())
4. Understanding relationships between variables

## 📈 Key Insights
- Loan amount varies based on income
- Different loan intents like personal, medical, education observed
- Basic patterns identified in dataset

- Add data visualization (graphs)
- Perform advanced analysis
- Build prediction model

This project helped in understanding the basics of data analysis using Python.
